App Info Movie++

This is an app for finding your favourite movies with relative ease. Just enter the title and then the magic happens!
After you've found the movie you've been looking for, you can add it to your "Favourite" list (and remove later, of course), so you can access it fast and easily ;)!

API

All the information about movies in this app is obtained via requests to the OMDb API.
Check them out!

Link:
http://www.omdbapi.com/

Implementation

This web app was implemented using Angular 9 and TypeScript. Also most of the component used in this app are from Angular Material UI library.
